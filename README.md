# 🏢 Office Management System

A complete Office Management System built with Node.js, Express, MongoDB, and EJS. Features department and employee management, self-referencing supervisors, server-side pagination, search/filters, and external location API integration.

## ✨ Features

- **Department Management** - Create, Read, Update, Delete departments
- **Employee Management** - Full CRUD operations for employees
- **Self-Referencing Supervisors** - Employees can report to other employees
- **Server-Side Pagination** - Efficient data loading with page navigation
- **Search & Filters** - Search by name/email, filter by department or job title
- **External API Integration** - Dynamic country/state/city selection using CountriesNow API
- **Modern UI** - Dark theme with glass morphism effects using Tailwind CSS
- **RESTful API** - Complete API endpoints for all operations

## 🛠️ Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose ODM
- **Frontend**: EJS templating, Tailwind CSS
- **External API**: CountriesNow API
- **Package Manager**: pnpm

## 📋 Prerequisites

- Node.js (v16 or higher)
- MongoDB (v6 or higher)
- pnpm (v7 or higher)
